A `Discord`_ bot that works.

Requirements
------------

Feynmanium requires:

-  `Python`_ v3.9 or later
-  `Poetry`_ v1 or later

Installation
------------

Use the package manager `Poetry`_ to install dependencies.

::

   poetry install

Usage
-----

::

   python .
