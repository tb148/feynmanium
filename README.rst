Feynmanium
==========

|Code style: black|

A `Discord`_ bot that works.

Requirements
------------

Feynmanium requires:

-  `Python`_ v3.10 or later
-  `Poetry`_ v1 or later

Installation
------------

Use the package manager `Poetry`_ to install dependencies.

::

   poetry install

Usage
-----

::

   python .

.. _Python: https://python.org/
.. _Poetry: https://python-poetry.org/
