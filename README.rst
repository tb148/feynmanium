A `Discord`_ bot that works.

************
Requirements
************

Feynmanium requires `Python`_ v3.10 or later.

************
Installation
************

Run the following command to install Feynmanium:

::

   pip install feynmanium

*****
Usage
*****

Run the following commmand to run Feynmanium

::

   feynmanium <token>
